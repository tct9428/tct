Below is a complete guide on how to automate messages using the `sched` command.

_Note: If using these commands inside a specific group, the bot defaults to that group. If sending from DM/Private Chat, you must specify the target group using the extension format (e.g., `jd:groupName` or `jd:123456@g.us`)._

*1. Create a Daily Recurring Schedule*

> â€¢ _Explanation:_ Sends "Good Morning Team!" every day at 8:30 AM.

*2. Create a One-Time Schedule (Specific Date)*

`sched add YYYY-MM-DD|HH:mm <message>`

> â€¢ *Example:* `sched add 2025-12-25|00:00 Merry Christmas!`
> â€¢ _Explanation:_ Sends the message exactly at midnight on Christmas, then stops.

*3. Create a One-Time Schedule (Next Occurrence)*

> â€¢ *Example:* `sched once 14:00 Remind me to call John`
> â€¢ _Explanation:_ If it is currently 13:00, this runs today at 14:00. If it is 15:00, it runs tomorrow at 14:00. It runs only once.

*4. Schedule by Relative Time (Countdown)*

`sched in <number> <unit> <message>`

> â€¢ *Example:* `sched in 45 m Laundry is done`
> â€¢ _Explanation:_ Sends the notification exactly 45 minutes from now.
> â€¢ *Example:* `sched in 2 d Follow up on report`
