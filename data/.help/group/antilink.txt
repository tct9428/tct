*3. Configuration & Limits*
Customize how strict the bot is. You can apply these settings to aliases too.

*Set Warning Limit*
How many links a user can send before getting kicked.
`.atl warn <number>`
> â€¢ *Example:* `.atl warn 3 jd:work-groups`

*Set Warning Message*
Customize the text. Use `{user}`, `{current}`, and `{limit}` as placeholders.
`.atl msg <text>`
> â€¢ *Example:* `.atl msg {user}, stop links! Warning {current}/{limit}.`

*Set Kick Message*
The message sent right before a user is removed.
`.atl kickmsg <text>`
> â€¢ *Example:* `.atl kickmsg {user} was removed for spamming links.`
