This module allows you to perform an HTTP GET request to any URL and retrieve the content directly in WhatsApp.
It automatically detects the content type and sends it as the appropriate media (Image, Video) or as a Document (HTML, JSON, Source Code).

_Note: This command is restricted to the Bot Owner and Sudo users._
