❌ *Tic-Tac-Toe Manual*

Play the classic X vs O game directly in your WhatsApp group.
This module handles game states, turns, timeouts, and ranking statistics.

_Note: This game requires two players._

✦═════════◆═════════✦

*1. Starting & Joining*
Challenge the group to a match.

`.ttt` or `.tictactoe`
> • _Explanation:_ Opens a new game lobby.
> • _Action:_ The bot waits for a second player to accept.

*Accepting a Challenge*
If someone else started a game, type this to join as Player 2.
`ready` or `.ready`
> • _Explanation:_ Accepts the pending challenge. The game starts immediately.

✦═════════◆═════════✦

*2. How to Play*
The game uses a standard 1-9 keypad layout.

*Making a Move*
Simply type the number of the spot you want to take.
`1` | `2` | `3`
`4` | `5` | `6`
`7` | `8` | `9`

> • *Example:* Type `5` to take the center spot.
> • _Note:_ You only need to type the number. No command prefix is needed during an active game.

✦═════════◆═════════✦

*3. Game Controls*
Managing the game session.

*Forfeit / End Game*
Type any of these words to stop the game immediately:
`cancel`, `end`, `stop`, `surrender`
> • _Explanation:_ Ends the current game. If a match was in progress, the one who surrendered loses.

*Timeouts*
> [cite_start]• Each player has *2 minutes* to make a move[cite: 22].
> • If the timer runs out, the other player wins automatically.

✦═════════◆═════════✦

*4. Rankings & Stats*
Check your win/loss record.

`.tttstats` or `.rank`
> • _Output:_ Displays your total wins, losses, and draws recorded in the database.
